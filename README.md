# MethodableHash

This is an extension of Hash, and enable to use key-value like method.

## Installation

Add this line to your application's Gemfile:

    gem 'methodable_hash'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install methodable_hash

## Usage

    require 'methodable_hash'
